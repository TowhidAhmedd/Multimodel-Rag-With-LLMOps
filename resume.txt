## VERSION A: RAG Architecture & LLM Optimization

**Production Multimodal RAG System with Advanced Retrieval & Prompt Engineering**

- Architected end-to-end RAG pipeline: multimodal ingestion (PDF/audio/video) → BAAI/bge-small-en-v1.5 embeddings (384-dim) → Pinecone vector search → Cross-Encoder re-ranking (ms-marco-MiniLM-L-6-v2) → Groq LLM inference (Llama 3.3-70B/3.1-8B/Mixtral 8x7B), reducing hallucinations by 35% and achieving 25% relevance improvement vs. single-stage retrieval
- Engineered intelligent prompt engineering pipeline with dynamic context assembly, confidence scoring (regex-extracted from LLM outputs), and strict grounding constraints in system prompts to eliminate out-of-distribution answers; validated quality across 100+ production queries with LangSmith tracing
- Implemented two-stage retrieval with semantic chunking (800-char, 150-overlap) and learned re-ranking to balance precision/recall; optimized latency from baseline 5s to <2s cached via TTL-based query caching and connection pooling (10 pool/20 overflow)
- Deployed production system on Render with Docker containerization and GitHub CI/CD; shipped with comprehensive observability (LangSmith tracing, structured JSON logging, confidence metrics) and 76 unit tests covering embedding quality, re-ranking calibration, and LLM output validation
